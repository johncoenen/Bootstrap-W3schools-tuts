Pills

Pills are similar to tabs, but styled differently. By default they are bordered on all sides, have white font color, and a blue background color.

Pills use the nav nav-pills classes and also use the active class to mark the current page.

Dynamic / Toggleable Tabs

Use the attribute data-toggle="tab" on each link to make the tabs toggleable.

Add the .tab-pane class using a unique ID for each tab.

Wrap these inside a <div> element with the class .tab-content.

Add a fade in/out effect by adding the .fade class to .tab-pane

Thanos is a fictional character appearing in American comic books published by Marvel Comics. Created by writer Mike Friedrich and writer-artist Jim Starlin, the character first appeared in Iron Man #55 (cover dated Feb. 1973).

MENU ONE

Thanos' first appearance was in Iron Man #55 (Feb. 1973), featuring a story by Jim Starlin that was scripted by Mike Friedrich. The storyline from that issue continued through Captain Marvel #25–33 (bi-monthly: March 1973 – Jan. 1974), Marvel Feature #12 (Nov. 1973), Daredevil #107 (Jan. 1974), and Avengers #125 (July 1974). He returned in an extended storyline that spanned Strange Tales #178-181 (Feb.–Aug. 1975), Warlock #9-11 (Oct. 1975 – Jan. 1976), Marvel Team Up #55 (March 1977), and the 1977 Annuals for Avengers and Marvel Two-in-One (Thanos does not actually appear until the end of Warlock #9).

MENU TWO

The character was revived in Silver Surfer vol. 3, #34 (Feb. 1990) and guest-starred until issue #59 (November 1991), while simultaneously appearing in The Thanos Quest #1–2 (Sept.–Oct. 1990) and Infinity Gauntlet #1-6 (July–Dec. 1991). After an appearance in Spider-Man #17 (Dec. 1991), Thanos had a recurring role in Warlock and the Infinity Watch #1-42 (Feb. 1992 – Aug. 1995).
