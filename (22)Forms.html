Bootstrap Forms

Default Settings

<input>, <textarea>, and <select> elements using the .form-control class have a width of 100%.

Form Layouts

The three types of form layouts used by Bootstrap are:

  • Vertical form = DEFAULT
  • Horizontal form
  • Inline form

Rules for the three form layouts:

  • Wrap labels and form controls in <div class="form-group">

    This is necessary for optimal spacing.

  • Add the .form-control class to any <input>, <textarea>, and <select> elements.

Inline Form

All elements are inline, left-aligned, and labels are alongside.

Note: This only applies to forms within view-ports that are at least 768px WIDE!

Additional rules for an inline form:

  • Add .form-inline with the <form> element

Horizontal Form

Labels align next to the horizontal input field for large and medium screen sizes.

Screens 767px and below will transform into a vertical form. The labels will move above each input field.

Additional Rules:

  • Use the .form-horizontal class with the <form> element
  • Use the .control-label class with all <label> elements

Use grid classes to align labels and groups of the form control for horizontal layouts.
