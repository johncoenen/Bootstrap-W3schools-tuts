Collapse is when you show or hide areas of content.

We use the .collapse class on an element to show or hide the content.

The data-toggle="collapse" attribute is attached to the <a> or the <button> element to show/hide the content.

Then the data-target="#id" attribute connects the button to the collapsible area.

When using <a> elements the href attribute can replace the data-target attribute.
