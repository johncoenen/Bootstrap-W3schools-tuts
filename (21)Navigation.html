Navigation Bars

Navigation bars will grow or shrink based on the size of the screen.

You can create a basic nav bar using: <nav class="navbar navbar-default">

Add Nav at Top of Page

Example:

Website Name/Title
  • Home
  • About
  • Products
  • Contact

Sizing Note:

These examples are for large screen formats, and are contained on a single line. Smaller screen formats will be dealt with in the final example.

Inverted Nav Bar

This is an alternative to the default navbar styling, and has a black background color.

Example:

Shoes for Feets
  • Home
  • Mens
  • Womens
  • Kids

Nav With Dropdown

Navigation bars can also use dropdown menus.

Example:

Musics
  • Home
  • Jazz
    • Bebop
    • Swing
    • Hard Bop
  • Rock n' Roll
  • Country

Nav Bar Right-Align

Use the .navbar-right class to right-align buttons.

Often seen examples of this are "Sign Up" and "Login" buttons that are separated from the main navigation areas..

Example:

HelloWebisite
  • Home
  • About
  • Contact
  •  SignUp
  •  Login

Nav Buttons

Use the .navbar-btn class inside the navbar to create a button.

Example:

Mish Mash
  • Home
  • Link
  • Link

Nav Forms

To create a form element in the nav, add the .navbar-form class to a form element. Then you can add additional inputs.

Using the .form-group class with a div container will add padding if there are more than one input.

Example:

Always Alvvays
  • Home
  • Page
  • Page
