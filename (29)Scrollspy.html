Scrollspy Plugin

The Scrollspy Plugin

This is used to auto update nav links based on scroll position.

Create a Scrollspy

Add data-spy="scroll" to the element that should be used as the scrollable area. This is often used on the <body> element.

Then add the data-target attribute using the value of the id/class name of the .navbar. This creates a connection with the navbar and the scrollable area.

Scrollspy elements must match the ID of the links inside the navbar list items. EX. <div id="section1"> matches <a href="#section1">

The data-offset attribute specifies the number of pixels to offset from top when calculating the position of scroll(THIS IS OPTIONAL).

The default is 10px. Use the data-offset method if the navbar active state is occuring too quickly.

Requires relative positioning: The element with data-spy="scroll" requires the CSS position property with a value set to relative to work correctly.
