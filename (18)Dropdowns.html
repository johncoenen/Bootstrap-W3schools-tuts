Dropdown Menus

Basic Dropdown

A menu that toggles when clicked and offers different values to chose from

The .dropdown class is used to create a dropdown menu.

Pair the dropdown menu with the .dropdown-toggle class and the attribute toggle="dropdown" to open the menu.

Use the .caret class to create the caret arrow icon, which indicates the menu can be dropped down.
