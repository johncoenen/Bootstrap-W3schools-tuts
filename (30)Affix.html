The Affix Plugin

This plugin allows an element to become affixed to an area on a page.

Often seen in navigation menus. Otherwise known as "sticky".

The plugin toggles the stickiness on/off based on the scroll position. When toggled the CSS position value changes from static to fixed.

Create Sticky Horizontal Nav Menu

<nav class="navbar navbar-inverse" data-spy="affix" data-offset-top="197">

Details

Add data-spy="affix" to the element that will become sticky.

Optionally, add data-offset-top|bottom to calculate the position of the scroll.

The Affix Plugin at Work

Toggles between three classes: .affix, .affix-top, and .affix-bottom

Each class represents a particular state. CSS properties are used to handle the actual positions. NOTE:There is an exception when using position:fixed on the .affix class. No CSS is needed in this case:

  • The plugin adds .affix-top or affix-bottom classes to indicate the element is in its top-most or bottom-most position. No CSS positioning is needed.
  • Scrolling past the affixed element triggers the actual affixing. The plugin replaces .affix-top or .affix-bottom classes to the .affix class and sets position: fixed. In this scenario CSS top or bottom properties are used to position the affixed element on the page.
  • When the bottom offset is defined: scrolling past replaces the .affix class using the .affix-bottom class. Because offsets are optional, setting one will require the use of CSS to position the element. In this case, add position:absolute when necessary.

In the Sticky Horizontal Nav example, the .affix class is added(position:fixed) to the <nav> element when the user scrolls to 197px from the top of the page. In this example the CSS property top is added with a value of zero on the .affix class. This ensures that the navbar stays at the top of the page when the user scrolls 197px from the top of the page.

Scollspy & Affix

How to implement the Affix plugin with the Scrollspy plugin:

Horizontal Navbar

<body data-spy="scroll" data-target=".navbar" data-offset="50">
<nav class="navbar navbar-inverse" data-spy="affix" data-offset-top="197">
...
</nav>

</body>

Vertical Side Navbar

<body data-spy="scroll" data-target="#myScrollspy" data-offset="15">
<nav class="nav nav-pills nav-stacked" data-spy="affix" data-offset-top="200">
...
</nav>

</body>
