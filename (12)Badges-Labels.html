Badges and Labels

Badges

Numerical indicators of how many items(associated with something ex. link)

The .badge class is used within a span element to create a badge.

Link with Badge

Articles 4

Emails 10000000

Button with Badge

Labels

Labels contain additional information.

Labels use the .label class as well as one of six contextual classes:

  • .label-default
  • .label-primary
  • .label-success
  • .label-info
  • .label-warning
  • .label-danger

To create a label, place .label in a <span> element.
