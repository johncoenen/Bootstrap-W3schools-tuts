Bootstrap Filters*

There is no component that allows filtering in Bootstrap. jQuery is used instead to filter elements.

Filtering with jQuery:
Use jQuery to loop through each table row and check if any text value matches the value of the search input. The toggle method hides the row display:none that does not match the search. The toLowerCase() method converts text to lower case and allows for case insensitive searching. In other words; john, John, and JOHN will return all cases of the name.
