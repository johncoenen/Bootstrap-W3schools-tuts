List Groups

An unordered list with list items is a basic example of a list group.

The <ul> element is paired with the .list-group class to create the unordered list group. Then the <li> elements are paired with the .list-group-item class to create the list items.

Linked Items in a List Group

To create a list group with linked items, replace <ul> with a <div> element. Then use the <a> tag in place of the <li> tag.
