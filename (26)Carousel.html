Carousel Plugin

This plugin cycles through elements. ex. Slideshow of photos

Hot Tip: Plugins can be used individually. ex. Bootstrap's "carousel.js" file. All at once, ex. using "bootstrap.js" or "bootstrap.min.js" files.

Basic Carousel

The outermost <div>:

Carousels require you use an id for the controls to work.

the class="carousel" specifies that this <div> contains a carousel.

The .slide class adds a CSS transition and animation effect, which makes the items slide to the next/previous item.

The data-ride="carousel" attribute tells Bootstrap to begin animating the carousel immediately when the page loads.

The "Indicators" part:

The indicators are the little dots at the bottom of each slide. These indicate the number of slides and which slide is being viewed now.

The indicators are specified in an ordered list using the .carousel-indicators class.

The data-target attribute points to the id of the carousel.

The data-slide-to attribute specifies which slide to go to, when clicking the specific dot.

The "Wrapper for slides" part

The slides are specified in a <div> with the class .carousel-inner.

The content of each slide is defined in a <div> with thee class .item. This can be text or images.

The .active class needs to be added to one of the slides or the carousel WILL NOT be visable.

The "left and right controls" part:

The code adds "left" and "right" buttons that allows the user to go back and forth between the slides manually.

The data-slide attribute accepts the keyword "prev" or "next", which alters the slide position relative to the current position.

Add Captions to Slides

Add <div class="carousel-caption"> within each <div class="item"> to create a caption for each slide.

Example

The Great Wave if Kanagawa

Hangs on many a dorm room wall!

Wanderer Above the Sea of Fog
