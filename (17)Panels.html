Panels

The .panel class is used to create a basic panel.

The content inside a panel uses the .panel-body class.

Panel Group

If you want to combine many panels together, wrap a <div> with the .panel-group class.

NOTE: The .panel-group class clears the bottom margin of each panel.

  • .panel-primary
  • .panel-success
  • .panel-info
  • .panel-warning
