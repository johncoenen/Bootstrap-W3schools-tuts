  • textarea
  • checkbox
  • radio
  • select

Form Control States

The following is a list of Bootstrap form control states:

  • INPUT FOCUS - Box outline is removed and box-shadow is applied on focus
  • DISABLED INPUTS - Disables an input field by adding a disabled attribute
  • DISABLED FIELDSETS - Disables all controls in a fieldset by adding a disabled attribute
  • READONLY INPUTS - Prevents user input by adding a readonly attribute to an input
  • VALIDATION STATES - With Bootstrap, validation styles for error, warning, and success messages can be used. Add .has-warning, .has-error, or .has-success to the parent element to implement styles.
  • INPUT - Add feedback icons by using the .has-feedback class and an icon
  • HIDDEN LABELS - Use a .sr-only class for non-visable labels
