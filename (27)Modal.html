The Modal Plugin

This is a plugin that is a dialog/popup window that displays on top of the current page.

Creating the Modal:

The "Trigger" part:

To trigger the modal window a button or link must be used.

This button/link should include the two data-* attributes:

  • data-toggle="modal" opens the window
  • data-target="#myModal" points to the id of the modal
The "Modal" part

The parent <div> of the modal must have an ID that is the same as the value of the data-target attribute used to trigger the modal.

The .modal class identifies the content of the <div> as a modal and gives it focus.

The .fade class adds a transition effect which fades the modal in/out. Just remove this class if undesired.

The attribute role="dialog" improves accessibility for users with screen readers.

The .modal-dialog class sets the proper width and margin of the modal.

The "Modal Content" part:

The <div> with the class="modal-content" styles the modal. Inside this <div> the modal's header, body, and footer.

The .modal-header class is used to define the style for the header of the modal. The <button> inside the header has a data-dismiss="modal" attribute which closes the modal if you click on it. The .close class styles the close button, and the .modal-title class styles the header with a proper line-height/

The .modal-body class is used to define the style for the body of the modal. Add any HTML markup here; paragraphs, images, videos, etc.

The .modal-footer class is used to define the style for the footer of the modal. Note that this area is right aligned by default.
