Popover Plugin

The Bootstrap Popover Plugin

This is similar to tooltips.

It is a pop-up box that appears when a user clicks on an element.

The main difference from tooltips is that a popover can contain much more content.

Create a Popover

Add the data-toggle="popover" attribute to an element.

Use the title attribute to specify the header text of the popover.

Use the data-content attribute to specify the text that will display inside the body of the popover.

Example:

<a href="#" data-toggle="popover" title="Popover Header Text" data-content="Some content...inside the popover.">Toggle Popover</a>

Note: Popovers are initialized using jQuery: select the specified element and call the popover() method.

Example

This is some code that will enable all popovers in a document.

<script>
$(document)ready(funtion(){
$('[data-toggle="popover"]').popover();
});
</script>

Positioning Popovers

Default popover style positions the popover on the right side of an element.

Use the data-placement attribute to set the position of the popover.

This can be on top, bottom, left, or right of the element.

Tip: "auto" is another data-placement attribute. This lets the browser decide the position of the popover. If the value is something like "auto left"m this will display on the left if possible, otherwise it will display on the right.

Closing Popovers

The default action for closing a popover is by clicking again on the trigger. It is also possible to use the data-trigger="focus" attribute to close the popover whenever an area outside the element is clicked.

example:
<a href="#" title="Dismissible popover" data-toggle="popover"
data-trigger="focus" data-content="Click anywhere in the document to close this popover">Click Me</a>

Tip: You can also display the popover using a hover event by giving the data-trigger attribute the value of hover.
